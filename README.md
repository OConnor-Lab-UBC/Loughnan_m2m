# Micro2Macro

### Repo started fall 2025 by D. Loughnan
Aim: We want to develop a modelling framework in which we can link microbial diversity to macrobial diversity change in eelgrass meadows in British Columbia, Canada.

* data: contains raw data files
* docs: contains meeting notes and project documents 
* figures: saved figures of data and model output
* input: modified, cleaning, or compiled data files used in subsequent analyses
* output: model output, excluding .Rda files that are not pushed to Github due to their size
* rcode: R code used in this analysis
* stan: stan code used for Bayesian modeling
